# EasyTier NeoUptime
一个基于Typescript+Express重写的EasyTier Uptime Monitor

## 功能特性

- 🏥 **健康监控**: 实时监控 EasyTier 节点的健康状态
- 📊 **数据统计**: 提供详细的运行时间和响应时间统计
- 🔧 **实例管理**: 管理多个 EasyTier 实例
- 🌐 **Web界面**: 直观的 Web 管理界面
- 🚨 **告警系统**: 支持健康状态异常告警
- 📈 **图表展示**: 可视化展示监控数据
- 🔒 **权限管理**: 支持用户权限管理(Github OAuth)
## 快速开始

### 环境要求

- **Node.js**: 20+
- **pnpm**: 10+
